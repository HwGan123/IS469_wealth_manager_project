Agents
◎
Market Context
Gathers live earnings, ratings, and news via MCP web scraper. Grounds downstream agents with real-time market data.
MCPNewsAPIDuckDuckGo
Writes: live_data_context, market_context
▦
Sentiment Agent
Scores news headlines per ticker using Llama-3.2-3B-Instruct. Aggregates into portfolio-level and per-ticker sentiment scores.
Llama-3.2-3BHuggingFacelocal
Reads: tickers, live_data_context
Writes: sentiment_score, sentiment_summary
⊡
Investment Analyst
Retrieves 10-K filing chunks from ChromaDB (RAG), combines with market context and sentiment to synthesise an investment thesis.
ChromaDBRAGGPT-4o-mini
Reads: tickers, sentiment_score, live_data_context
Writes: retrieved_context, draft_report
◈
Auditor Agent
LLM-as-Judge. Cross-checks every factual claim against RAG context. Computes RAGAS metrics. Flags hallucinations and triggers self-correction.
RAGASLLM-judgeGPT-4o-mini
Reads: draft_report, retrieved_context
Writes: audit_score, is_hallucinating, ragas_metrics
▤
Report Generator
Synthesises the verified draft, sentiment summary, and audit log into the final client-facing markdown report.
GPT-4o-miniMarkdown
Reads: draft_report, audit_findings, sentiment_summary
Writes: final_report
